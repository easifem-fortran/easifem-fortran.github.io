# SetColumn

This method sets the column of the CSRMatrix.

Calling example

- `SetColumn(obj, icolumn, value(:,:))`
- `SetColumn(obj, nodenum, idof, value(:,:))`
- `SetColumn(obj, icolumn, value)`
- `SetColumn(obj, nodenum, idof, value)`
- `SetColumn(obj, nodenum, ivar, idof, value)`
- `SetColumn(obj, nodenum, ivar, idof, value(:))`
- `SetColumn(obj, nodenum, ivar, spacecompo, timecompo, value)`
- `SetColumn(obj, nodenum, ivar, spacecompo, timecompo, value(:))`
- `SetColumn(obj, nodenum, ivar, spacecompo, timecompo(:), value)`
- `SetColumn(obj, nodenum, ivar, spacecompo, timecompo(:), value(:))`
- `SetColumn(obj, nodenum, ivar, spacecompo(:), timecompo, value)`
- `SetColumn(obj, nodenum, ivar, spacecompo(:), timecompo, value(:))`
- `SetColumn(obj, nodenum(:), ivar, spacecompo, timecompo(:), value)`
- `SetColumn(obj, nodenum(:), ivar, spacecompo, timecompo(:), value(:))`
- `SetColumn(obj, nodenum(:), ivar, spacecompo(:), timecompo, value)`
- `SetColumn(obj, nodenum(:), ivar, spacecompo(:), timecompo, value(:))`
